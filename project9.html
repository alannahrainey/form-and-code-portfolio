    • Reflection

    The process of creating an interactive kaleidoscope effect using p5.js and a webcam was an exploration of core features such as webcam integration and graphics programming. Using p5.js, the project featured real-time adjustments to the visual effect based on mouse position, introducing a layer of interactivity that enhanced user engagement. Challenges such as optimizing performance for smooth rendering and ensuring cross-browser compatibility were addressed through iterative testing and feedback, especially when using the webcam as an accessible feature on the platform. This project showed new opportunities for video and canvas manipulations in a web environment but also highlighted the creative potential of combining the viewer as an element in the design with dynamic visual effects to produce compelling and immersive online experiences.
